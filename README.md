# Cartorio_EBAC
Projeto desenvolvido durante o curso de Tecnologia da Informação da EBAC, com foco na prática de fundamentos de programação em C++.

## Descrição:
Aplicação desenvolvida para simular operações básicas de um cartório, permitindo o cadastro e consulta de registros via terminal.

## Funcionalidades
- Cadastro de registros
- Consulta de dados cadastrados

## Tecnologias:
C++

## Conceitos aplicados:
- Estrutura de dados simples
- Entrada e saída de dados no terminal
- Lógica de programação aplicada
- Organização e estruturação de código

## Como executar
1. Clone o repositório:
git clone https://github.com/JacquelineTavaresDev/Cartorio_EBAC
2. Acesse a pasta do projeto:
cd Cartorio_EBAC
3. Compile o arquivo principal:
g++ main.cpp -o cartorio

## Executar no Windows
1. Prompt de Comando (CMD):
cartorio.exe
2. PowerShell:
.\cartorio.exe
## OBS: É necessário ter o compilador C++ (g++) instalado na máquina.

## Sobre o projeto:
Projeto desenvolvido para consolidar conhecimentos em lógica de programação e manipulação de dados utilizando C++.
